# Relacion-entre-factores-ambientales-y-la-demanda-hospitalaria-por-las-IRA-en-Panama-2017-2023-
Este estudio analizo la relación entre factores meteorológicos, contaminantes atmosféricos e incidencia de Infecciones Respiratorias Agudas (IRA) durante el 2017–2023, así como su impacto en la demanda hospitalaria de medicamentos, equipos médicos y atenciones en urgencias, aplicando modelos como:  GWR, MGWR, ST-GAM, BAM, XGBoost, BART, Regression Ridge y Poisson GLM.
